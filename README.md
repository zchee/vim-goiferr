vim-goiferr
===========
Automatically inserts idiomatic error handling to given Go source code for vim.

Usage:
------

```vim
:GoIferr
```

Requirements:
-------------

```bash
go get -u github.com/motemen/go-iferr/cmd/goiferr
```

Install:
--------

```vim
NeoBundle 'zchee/vim-goiferr'
# or
Plug 'zchee/vim-goiferr'
```
